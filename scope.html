Scope

  • The scope is binding part between the HTML(view) and Controller
  • The scope is an object with available properties and methods
  • The scope is available for both the view and controller

Understanding the Scope

If we consider an AngularJs application to consist of:

  • View: which is the HTML
  • Model: which is data available for the curent view
  • Controller: which is Javascript function that makes/changes/removes/controls data

The scope is model

Root Scope

  • All aplication have a $rootScope which is the scope created on the HTML element that contains the ng-app directive
  • The rootScope is available in the entire application
  • If a variable has the same name in both the current scope and in the rootScope, the application uses the one in the current scope
