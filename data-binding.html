Lession 5 - Data Binding

Data-binding in Angularjs is synchronization between the model and the view

Data Module

AngularJs Application usually have a data model. The data model is a collection of data variable for the application

HTML view

  • The HTML container where the AngularJs application is displayed, is called the view.
  • The view access to the model, and there are several ways of displaying model data in the view.
  • You can use the ng-bind directive, which will bind the innerHTML of the element to the specified model property

AngularJS Controller

Because of the immediate synchronization of the model and the view, the controller can be completely separated from the view, and simply concentrate on the model data. Thanks to the data binding in AngularJS, the view will reflect any changes made in the controller.
