Lesson 2 - Directives

  • Angular Js lets you extend HTML attributes called Driectives
  • Angular Js has a set of built-in directives which offers functionality to your application
  • Angular Js also lets define your own directives

Angular js directives are extend HTML attributes with the prefix ng-

  • The ng-app directive initializes an Angular Js application
  • The ng-init directive initializes application data
  • The ng-model directive binds the value of HTML controls (input, textarea, select) to application data

Creat new directives

  • In addition to all of the built-in AngularJs directives, you can creat your own directives
  • New directive using .directive function
  • To invoke new directive, make an HTML element with the same tag name as the new directive
  • When naming a directive, you must use camel case name
